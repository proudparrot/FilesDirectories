Instructions on Compiling and Running Program
1) add all the files into a single directory
  a. The required files to successfully compile are:
    1. main.c
    2. mainmenu.h
    3. submenu.h
    4. findlargest.h
    5. findsmallest.h
    6. findprefix.h
    7. processdir.h
    8. createlist.h
    9. filemaker.h
2) Once inside the directory with the files above
    type in the command line: gcc --std=gnu99 -o movies_by_year main.c
3) an executable file (named: movies_by_year) will be created in this directory
4) to run this executable file type in the command line: ./movies_by_year
5) The program should be up and running at this point.
